# Math Curve Loaders

A gallery of **mathematical curve–based loading animations**: parameterized curves drive the motion, with formulas, live controls, and exportable React component code in the detail view.

**Source:** [Paidax01/math-curve-loaders](https://github.com/Paidax01/math-curve-loaders) — the original is a dependency-free HTML, CSS, and JavaScript gallery. This repository is a **React** reimplementation of that project.
